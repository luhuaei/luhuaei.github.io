# ABI 是什么?

根据 [wiki](https://en.wikipedia.org/wiki/Application_binary_interface) `ABI` 是 `Application Binary Interface` 的简写.

在硬件和最底层的软件之间使用架构指令集(instruction set architecture)进行抽象. 一般情况下, 操作系统都会将`I/O, alloc`等底层的封装好.

指令集和操作系统结合起来,提供给程序的接口就被称为`ABI(application binary interface)`
