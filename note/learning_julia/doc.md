2. 定位为一门数据科学的语言, 内置了一些常用的常量, 如`pi`; 并支持重新赋值.奇怪点在于, 如果你已经在前面已经使用过这个常量,将不能再次赋值.这个跟`R`中不同, `R`中不管任何情境下也可以对`pi`再次赋值.

3. [1, 2, 3] * 3 代表的意思?
```
python> [1, 2, 3] * 3 ==> [1, 2, 3, 1, 2, 3, 1, 2, 3]
julia> [1, 2, 3] * 3 ==> [3, 6, 9]
R> c(1, 2, 3) * 3 => [3, 6, 9]
```
